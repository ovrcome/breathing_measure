Purpose of this repo:

- To enable the user to collect data on breathing by running the 'breathing_measure.py' programme concurrently with a Shimmer device recording PPG data.
- To take the data collected from both sources, combine it and plot it.

Full instructions for running the programme and using the Shimmer have been written up on this confluence document:

https://ovrcome.atlassian.net/wiki/spaces/~7120204f8a0d96cea84511a9e72dcbd0d7a3fe/pages/836927502/Instructions+for+Shimmer+Data+Collection


Getting started:

- Install the libraries listed in 'requirements.txt'
- Follow the instructions in the Confluence Doc to record and combine the data
